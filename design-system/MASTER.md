# RemindersMac 设计系统

> 生成时间: 2026-03-16 17:27 CST
> 最后更新: 2026-03-16 20:15 CST

## 设计方向

**风格**: Dual Claymorphism（浅色奶油粘土 + 深色巧克力硅胶）
**调性**: 浅色模式温暖轻盈、圆润可爱；深色模式改为更像移动端 to-do app 的 3D 拟物粘土质感，带软硅胶、挤压感和糖果色高亮。

## 设计目标

- 浅色模式保持已完成的温暖奶油 Claymorphism。
- 深色模式升级为强视觉风格：暖巧克力背景、软硅胶材质、圆角饱满、拟物开关、夸张按钮、Dribbble 风格细节。
- 让深色面板更接近“移动端待办列表卡片”的构图与质感，同时保留当前功能结构。

### 深色模式组件特征

- 顶部输入区必须像“深深凹进去”的软硅胶槽。
- `+` 按钮必须是 chunky、触感厚实的实体按钮。
- toggle switch 必须显式呈现物理滑块。
- 任务卡像单独浮起的软胶片，圆角要更大。
- checkbox 为实体凹凸块，不是扁平 outline。
- 任务行增加一枚可爱 chunky 3D 猪头 icon。
- 时间文字使用 neon red，高对比跳出。
- 底部 clear 按钮是圆润 gummy pill，不再是普通文本按钮。

## 字体

- **标题 / UI 主字重**: SF Pro Display
- **正文 / 列表文字**: SF Pro Text

## 形状语言

- 主容器圆角: 26px
- 次级区域圆角: 20px
- 输入框 / 按钮圆角: 16px
- 小控件圆角: 12px
- 深色模式允许更夸张的大圆角与更厚重的实体体积

## 阴影系统

### 浅色模式
- 主面板: `8px 8px 18px rgba(188, 162, 136, 0.38)`
- 轻面板: `6px 6px 12px rgba(201, 176, 151, 0.30)`
- 浮层卡片: `12px 12px 24px rgba(180, 149, 120, 0.30)`

### 深色模式
- 主卡片外阴影: `14px 16px 32px rgba(10, 6, 6, 0.42)`
- 浮起按钮: `6px 8px 12px rgba(22, 10, 8, 0.35)`
- 内凹槽: `inset 6px 6px 10px rgba(24, 12, 10, 0.55)` + `inset -3px -3px 6px rgba(107, 74, 65, 0.16)`
- 软高光: `-3px -3px 8px rgba(255, 220, 210, 0.05)`

## 动效

- 微交互: 180ms ease-out
- 页面过渡: 260ms ease-in-out
- 按压效果: 缩放至 0.985，并减弱外阴影、增强内阴影
- toggle 切换要有滑块位移动画
- 遵循 prefers-reduced-motion

## 组件规范

### 浅色模式
- 保持现有温暖粘土视觉

### 深色模式
- 构图上更像移动端 to-do list 卡片
- 顶部 input + plus + toggle 形成明确的功能头部
- 列表中每一行是独立软块
- pig icon 必须简洁、可爱、块状，不使用 emoji
- 全局光照要统一，避免每个组件像不同材质

## 禁止事项

- 不使用冷黑 + 冷灰的普通暗色 dashboard 方案
- 不使用扁平化蓝黑配色
- 不使用过细描边破坏软质感
- 不让深色模式变成玻璃拟态或 cyberpunk 风格
- 不用 emoji 猪头，必须是统一风格的系统化图形表达
